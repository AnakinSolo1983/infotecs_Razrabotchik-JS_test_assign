# Тестовое задание --- Разработчик JS (React)

Тестовое задание выполнено в рамках отбора на позицию стажёра
«Разработчик JS».

## Описание проекта

Приложение представляет собой таблицу пользователей с возможностью:

-   сортировки по полям:
    -   ФИО
    -   возраст
    -   пол
    -   номер телефона\
-   фильтрации данных
-   постраничного отображения (pagination)
-   просмотра подробной информации о пользователе в модальном окне
-   изменения ширины колонок таблицы

Данные загружаются с публичного API:\
https://dummyjson.com/docs/users

Для выполнения HTTP-запросов используется **только Fetch API**.

## Функциональность

### Основные требования

-   Таблица шириной 100% экрана (но не более 1400px)
-   Колонки:
    -   Фамилия
    -   Имя
    -   Отчество
    -   Возраст
    -   Пол
    -   Номер телефона
    -   Email
    -   Страна
    -   Город
-   Сортировка по:
    -   ФИО
    -   возрасту
    -   полу
    -   номеру телефона
-   Сортировка имеет 3 состояния:
    -   по возрастанию
    -   по убыванию
    -   без сортировки
-   Получение данных через HTTP-запросы к API

## Работа с API

Используемые эндпоинты:

-   Получение пользователей\
    `GET https://dummyjson.com/users`

Сортировка реализуется через параметры запроса либо на клиентской
стороне с сохранением состояния сортировки.

Все запросы выполняются через `fetch()`.

Реализована обработка:

-   network errors
-   некорректных ответов сервера
-   состояний загрузки
